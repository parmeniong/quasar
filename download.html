INSTALL

NOTE: Quasar is currently available only for Linux and only for the x86_64 CPU architecture.

Installing QVM

To install Quasar binaries you first need to install QVM, the Quasar Version Manager.

To install QVM, visit QVM's installation page.

Installing Quasar

Once you have installed QVM, run the following command to install the latest version of Quasar:

qvm install latest

To check if Quasar was installed properly, run the following command:

quasar --version

Build from source

You can also build Quasar from source. You can find Quasar's source code in its github repository. You can also try to build it for other platforms but you might have to make changes or it might not even work at all.
